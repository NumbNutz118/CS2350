Using display, width, and margin properties

This section demonstrates a way to center that requires three style properties be considered.

  1. Add a style rule that sets the width property of the following ul element to 420px.
  2. Add a style rule that sets the margin property to auto on the following ul element.
  • display: block
  • width: * must have a specific width
  • margin-right and margin-left must be set the same. The easiest way to do this is margin: auto;

Notice that it centered the list including the bullets. We did not have to set the display property because the default display property of the ul is block.

There are several things to do on the following table. First we will add borders and padding.

  1. Add a style rule that sets the border property of the th and td elements. You choose a border width, border color, and border style. I have use a width of 2 pixels, color #000000, and style solid. Pick different values for each of these properties. Your table will look different than the one in the example.
  2. Add a style rule that sets the padding property of the td and th cells. You choose the padding value.
  3. Add a style rule that sets the border-collapse property of the table to collapse. Notice that this property needs to be used with the selector table (NOT th or td).

Now let's center the table.

  1. Add a style rule that sets the width property of the table to 60%.
  2. Add a style rule that sets the margin property of the table to auto.

Playing with the Background

In this section we are going to play with the following background properties.

  • background-color
  • background-image
  • background-position
  • background-repeat

All of these properties can be set at one time with the combined background property. There are some other properties that can be set. You can find these additional properties explained on w3schools reference on the CSS background Property.

To start with we want to make all the divs in this section big enough to see the background well. Notice how the class values all start with background followed by a number. You can use the CSS [attribute^=value] selector to select all elements with a class that begins with the same characters. In this case we would use the selector: [class^="background"] This will select all elements that have a class attribute beginning with background.

  1. Use the attribute selector to set the width to 600px and the height to 400px for all the background divs.
  2. Using this same selector, set the border property to 4px solid #3E3175.
  3. Using this same selector, set the margin-top property to 24px.
  4. Finally, set the margin-right and margin-left properties to auto. This will center the divs because we have already set a width. Notice how we could not use the combined margin property on this one or it would override out margin-top property that was previously set.

That is the end of the style we want to add to all these divs. Now we will select each div individually so we can make the background style for each one different. To do this use the following selectors: .background1, .background2, .background3, .background4,.background5, and .background6.

Notice how we can cover the background entirely, in just one spot, make a strip, or add a transparent texture. Solid background colors can be bold or harsh. The transparent textures allow you to keep the background color but adds texture and softens it.

  1. Using the following selectors: .background1, .background2, .background3, .background4,.background5, and .background6 add the background property listed in each div.

Notice how the name of the image file does not include the resources folder name. This is because the .css file is in the resources folder so when we want to access an image from the .css file, there is no need to change folders.

Using Float and Clear

Using float can create interesting effects. It can also create unwanted side effects. When using float pay particular attention to how it looks as you resize the browser window.

Notice how the class values all start with css followed by a number. You can use the CSS [attribute^=value] selector to select all elements with a class that begins with the same characters. In this case we would use the selector: [class^="css"] This will select all elements that have a class attribute beginning with css.

  1. Use the attribute selector to set the float property to left for all the css paragraphs.
  2. Using this same selector, set the margin property to 12px.

Notice how they are taken out of the normal flow of the HTML. The body element expands to contain all the content that is inside the body tags. But since all these paragraphs have their float property set, they are outside the normal flow and are no longer contained inside the body.

To fix this, we can add an element below the paragraphs and set the clear property to none. It is often a good idea to add such a clearing element after float is used. I have added an empty div just for this purpose.

  1. After you have seen the images spill out of the body, add a style rule that sets the clear property of the empty div to both.
  2. Verify the body extends to include this empty div. Since the div stays after the floating paragraphs (because of the clear property), the images also stay within the body.
