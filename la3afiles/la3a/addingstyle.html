Adding Style Rules

Each style block begins with a selector or a series of selectors. These selectors specify which tag or tags the style is to be applied to. After the selector(s) comes {}s. Within the {}s are one or more style rules.

Each style rule is made up of a property:value pair. The property must be one of the properties defined in CSS. You can find a list of the these properties on the CSS Reference from W3Schools.

Each property has specific values that are allowed for that property. Values come in a variety of types. Below is a table of the types of values that are possible.

Browser Style Sheets

Browsers have a style sheet. This is called the user agent style sheet. These provide the default style to be used whenever there is no other style specified. The default style rules vary depending on what browser is being used. Because of these variations some recommend that a reset style sheet be used. A reset style sheet gets rid of much of the existing style and lets the web developer start with a clean slate no matter which browser is being used.

Navigation Lists

A common way to create navigation is to use a list. But most navigation does not look like a list. To get a list to work for navigation, the styles of the ul and li tags are changed.
