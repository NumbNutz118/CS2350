The image above is the css box model. It has 4 parts to it. At the center is your content. This could be words, a paragraph, or even a photo. The second layer is padding. This is the blank space between the content and the next layer the border. Border's aren't needed but borders can be different sizes, different looks, and different colors to add variety to your webpages. The final layer is the margin. This is the blank space between the border, and either the edge of the webpage or more content on the page like paragraphs or other photos.

Padding

Padding Same Size 64 pixels

All the padding is the same size here. As you can see on all sides of the text the border is the same distance away. This is the padding. Sizes can change depending on how you want your content to be displayed.

Top and Bottom different than Left and Right

In this example the top and bottom are the same size padding at 12 pixels, and the right and left are the same at 64 pixels. You can see that the top and bottom are much smaller compared to the last example.

All sides different

In this example all the sides are different sized padding. You can see that the right side is the largest followed by the bottom, with the left side barely being bigger than the top. These 3 examples show how padding can change how your webpage looks with certain content.

Margins

No margin changes

In this examples there are no margin changes. It's hard to tell what no changes look like without something to compare it to so you'll see in the next examples what the margin changes.

Left margin at 50 pixels

In this example you see that the left sides of the border is pushed in a little bit. That's the left margin. The distance between the border and the edge of the webpage.

Left margin at 160

In this final example you can see that the left margin is even bigger. Using this you can move things where you want them to go. Of course you can change the top bottom and right margins too, but this example is only showing the left margin.

You can also see in these margin examples that the text is really close to the border. That's where you can combine padding and margin to get you content to where you want.

Hovering

Hovering is really cool in css. This is when you just hover the mouse over content. No clicking just hovering. You can change color of the background, scale, rotate, basically anything to the content. You can also adjust how long it takes for the transition to happen. Below are some examples of what you can do with hovering.

No change

There is no change to hovering in this example. This is just what the default hover is. Nothing.

Color change

In this example when you hover over this content you'll see that the color of the background changes. And you can also see it takes 4 seconds to do so which is really cool to do.

Size change

In this example when you hover over this content you'll see that the width changes. You can add this if you just want some style to your page. One issue with this is you see that the text moves with it, so it's hard to read along when using text.

Position

Relative Position

Relative positioning is position based off of it's normal position. So where it is placed in the html file is the normal position, and any adjustments you make will change it based on that position. It's hard to tell but the image below was moved 40 pixels from the left, and 20 pixels from the top from where it originally started. This effect is similar to margin changing.

Absolute Position

Absolute positioning is based off of it's closest ancestor. But if the content does not have an ancestor it uses the document body as the positioning. So as you can see the image is at the top instead of where it should be.

Fixed Position

Fixed positioning is based off of the viewport, or the window. You must tell the image where to go on the screen and it will stay at that position. You can see the image is staying in the top right no matter if you scroll or what you do.

One thing you can see with the absolute and fixed positions are the images can cover up other content when scrolling or resizing the screen. So be careful with these positionings to make sure they don't block content.

Shadows

Shadows are a really cool part of websites that can make things stand out on the webpage. There are 3 important parts to shadows which are the horizontal property, the vertical property, and the blur property.

Horizontal Property

The horizontal property is how far the shadow goes left to right. Default numbers cause the shadow to go right. With a negative value that will cause the value to go left.

Vertical Property

The vertical property is how far the shadow goes up to down. Default numbers cause the shadow to go down. With a negative value that will cause the value to go up. Combined with the horizontal property the light will appear default to be top left. You can change that with a mix of negative numbers.

Blur Property

The blur property is how blurry the shadow is. A 0px blur value will be no blue at all. It will be an exact copy of the content at a different position based on the other properties. A bigger value will be more blur. So depending on how blurry you want the shadow to look, this value will be changed.
